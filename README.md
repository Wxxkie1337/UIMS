# Urban Issues Monitoring System

Telegram-бот для приема и модерации обращений жителей по городским проблемам.

## Что умеет
- создание обращений с категорией, описанием, фото и геопозицией;
- просмотр своих обращений и их статусов;
- модерация обращений (принять/отклонить с причиной).

## Стек
- Python 3.13
- aiogram
- SQLite (aiosqlite)

## Запуск
```bash
python src/bot.py
```

## Структура
- `src/handlers/` — обработчики бота;
- `src/keyboards/` — клавиатуры;
- `src/db/` — слой работы с базой данных.
